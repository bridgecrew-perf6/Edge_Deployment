# Edge_Deployment
Tutorial of deployment of deep neural networks on various types of edge devices

## Target devices

1. Pixel 3 XL
2. Raspberry Pi 4
3. Nvidia Jetson Nano b01/ Xavier NX
4. Coral edge TPU
5. Intel Neural Compute Stick2 (NCS2)
